## 🚦 Getting Started: Bot Setup

### 1. Create Your Bot Identity

1. Visit [Discord Developer Portal](https://discord.com/developers/applications)
2. Click "New Application"
3. Navigate to "Bot" > "Add Bot"

<Callout emoji="🔑">
  Save your bot token immediately! This is your bot's password - never share it or commit it to GitHub.
</Callout>

### 2. Install Dependencies

```bash
pnpm init -y
pnpm install discord.js @discordjs/rest discord-api-types dotenv
```

## 💻 Bot Anatomy 101

Create `bot.js`:

```javascript
require('dotenv').config();
const { Client, GatewayIntentBits } = require('discord.js');

const client = new Client({
  intents: [
    GatewayIntentBits.Guilds,
    GatewayIntentBits.GuildMessages,
    GatewayIntentBits.MessageContent
  ]
});

client.on('ready', () => {
  console.log(`🤖 ${client.user.tag} is ready for action!`);
});

client.on('messageCreate', async message => {
  if (message.content === '!ping') {
    await message.reply('🏓 Pong!');
  }
});

client.login(process.env.DISCORD_TOKEN);
```

<Callout emoji="💡">
  Create a `.env` file with DISCORD_TOKEN=your-bot-token to keep credentials safe
</Callout>

## 🎭 Adding Personality

### Slash Commands Setup

```javascript
const { SlashCommandBuilder } = require('discord.js');

module.exports = {
  data: new SlashCommandBuilder()
    .setName('highfive')
    .setDescription('Virtual high five!'),
  async execute(interaction) {
    await interaction.reply('🖐️ Up top! *smack*');
  }
};
```

### Command Deployment

Create `deploy-commands.js`:

```javascript
const { REST, Routes } = require('discord.js');
const commands = [require('./commands/highfive.js').data.toJSON()];

const rest = new REST({ version: '10' }).setToken(process.env.DISCORD_TOKEN);

(async () => {
  try {
    console.log('🚀 Deploying commands...');
    await rest.put(
      Routes.applicationCommands(process.env.CLIENT_ID),
      { body: commands }
    );
    console.log('✅ Commands deployed!');
  } catch (error) {
    console.error('❌ Deployment failed:', error);
  }
})();
```

## 🚀 Advanced Features

### Interactive Buttons

```javascript
const { ActionRowBuilder, ButtonBuilder, ButtonStyle } = require('discord.js');

client.on('interactionCreate', async interaction => {
  if (!interaction.isCommand()) return;

  if (interaction.commandName === 'quiz') {
    const row = new ActionRowBuilder()
      .addComponents(
        new ButtonBuilder()
          .setCustomId('true_btn')
          .setLabel('True')
          .setStyle(ButtonStyle.Success),
        new ButtonBuilder()
          .setCustomId('false_btn')
          .setLabel('False')
          .setStyle(ButtonStyle.Danger)
      );

    await interaction.reply({
      content: 'Is JavaScript the best language?',
      components: [row]
    });
  }
});
```

### Scheduled Tasks

```javascript
const cron = require('node-cron');

cron.schedule('0 9 * * *', () => {

  channel.send('🌞 Good morning! Time to code!');
});
```
